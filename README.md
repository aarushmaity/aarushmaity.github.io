### What I'm building

**[Milwaukee Traffic Risk Prediction System](https://github.com/aarushmaity/milwaukee-traffic-risk)**
An ML platform that scores every intersection in Milwaukee weekly for crash risk — built to guide real infrastructure decisions. Trained on Wisconsin DOT crash data (2019–2022) using XGBoost with SHAP explainability so city planners can see *why* a corridor is flagged, not just *that* it is.

> Funded through the **Palantir Valley Forge Grants Fellowship**.
